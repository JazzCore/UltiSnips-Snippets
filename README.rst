UltiSnips-Snippets
==================

My custom UltiSnips snippets, mainly taken from SnipMate sources.

Installation
------------

.. code-block:: vim

    Bundle 'JazzCore/ultisnips-snippets'

Add to ``.vimrc``:

.. code-block:: vim

    let g:UltiSnipsSnippetDirectories = ["UltiSnips", "ultisnips-snippets"]
